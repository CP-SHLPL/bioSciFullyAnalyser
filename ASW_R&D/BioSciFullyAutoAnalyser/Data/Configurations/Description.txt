This Folder will contain the specific configurations for the tables like Indexes, Constraints, Relationships, etc. 
This will be used by the DbContext to apply the configurations to the database when it is being created or updated. 
Each configuration will be in a separate file named after the table it is configuring. 
For example, if we have a table called "TblProducts", we will have a file called "TblProductsConfiguration.cs" in this folder that contains the configurations for the "TblProducts" table.
